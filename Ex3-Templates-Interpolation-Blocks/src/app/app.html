Interpolation Examples

Templates can include interpolation expressions using double {{curly}} braces to bind data from the component class to the template.

Example 1: We can display numOne using interpolation: {{numOne}} ... we can also display numTwo using interpolation: {{numTwo}}

Example 2: We can also perform arithmetic operations: {{numOne}} + {{numTwo}} = {{numOne + numTwo}}


Block Directives Examples

Templates can also include block directives for conditional rendering and loops.

@if (showMessage()) {

The showMessage signal is true, so this message is displayed.

} @else {

The showMessage signal is false, so this message is hidden.

}
